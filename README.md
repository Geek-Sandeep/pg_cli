# Install Guide:

```sh
# Install npm dependencies
npm install;

# link the package
npm link;
npm link pg;
```

# Commands

## Pg Command

```shell
$ pg --help
Usage: pg [options] [command]

Postgres CLI

Options:
  -V, --version              output the version number
  -h, --help                 display help for command

Commands:
  create:db <database-name>  create the postgres database
  drop:db <database-name>    drop the postgres database
  help [command]             display help for command
```
